
Shadow & Flame - LUNC Game Starter Project (Godot 4.x)
------------------------------------------------------

Folder Structure:
- scenes/      : Godot scene files (.tscn)
- scripts/     : GDScript logic
- assets/      : Images, audio, fonts, UI elements
- config/      : Game settings or data files
- narrative/   : Dialogue, lore, puzzle text
- levels/      : Level layout files
- addons/      : Any installed Godot plugins

Ready to start building!
